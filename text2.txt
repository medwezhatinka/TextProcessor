Объект класса NewBook может быть создан двумя способами, вызывающими один из конструкторов: 
NewBook tips1; // объявление 
tips1 = new NewBook();// инициализация 
NewBook tips2 = new NewBook("Java2", "Ноутон", 9.f); 
Оператор new вызывает конструктор, поэтому в круглых скобках могут стоять аргументы, передаваемые конструктору. 
Если конструктор в классе не определен, Java предоставляет конструктор по умолчанию, который инициализирует объект значениями по умол-
чанию. Если же конструктор с параметрами определен, то конструктор по 
умолчанию становится недоступным и для его вызова необходимо явное 
объявление такого конструктора. 
В следующем примере объявлен класс Locate c двумя полями (атрибутами), конструктором и методами для инициализации и извлечения зна-
чений атрибутов.
